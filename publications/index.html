    Abs DOI

    For the Autonomous Maritime Navigation System, LiDAR 3D object detection is crucial for waterborne environment perception. However, complex dynamics and multi-scale objects, particularly small vessels, present significant accuracy challenges for current algorithms. To address these limitations, this study introduces point cloud intensity into the PointPillar detection network and proposes a novel 3D object detection method named Intensity-PointPillar. By incorporating average pooling and attention pooling into the encoding network, the method fully considers the local geometric details and point cloud intensity information within each pillar module, enhancing the feature representation capability of each pillar and thereby improving the model’s object detection performance. Additionally, a multi-branch detection head is introduced, along with a point cloud intensity loss function, to mitigate the inconsistency between the object classification branch and the position regression branch. The proposed method is trained and evaluated using two real-world ship LiDAR point cloud datasets. Experimental results demonstrate that the Intensity-PointPillar achieves significantly higher detection accuracy, with a mean Average Precision improvement of 31.6 % compared to the original network, thereby validating the effectiveness of the proposed method. © 2025 The Authors.
